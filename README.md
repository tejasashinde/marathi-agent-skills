# Marathi Agent Skills

This repository contains a small set of focused agent skills for Marathi-language work. Each skill lives in `skills/<skill-name>/` and follows the local skill format used in this repo.

## Repository Structure

- `skills/` - skill packages
- `README.md` - repository overview and skill catalog

## Skill Catalog

| Skill | Location | Short Description |
|---|---|---|
| `marathi-proofreader` | [`skills/marathi-proofreader/SKILL.md`](skills/marathi-proofreader/SKILL.md) | Proofreads Marathi text and Marathi documents while preserving voice, tone, and meaning. |
| `translating-english-marathi` | [`skills/translating-english-marathi/SKILL.md`](skills/translating-english-marathi/SKILL.md) | Translates between English and Marathi while preserving meaning, tone, and structure. |
| `marathi-transliterator` | [`skills/marathi-transliterator/SKILL.md`](skills/marathi-transliterator/SKILL.md) | Transliterates Marathi between Devanagari and Roman script without translating meaning. |
| `marathi-doc-explainer` | [`skills/marathi-doc-explainer/SKILL.md`](skills/marathi-doc-explainer/SKILL.md) | Explains Marathi documents in simple, structured language without changing the original meaning. |
| `marathi-recipe-generator` | [`skills/marathi-recipe-generator/SKILL.md`](skills/marathi-recipe-generator/SKILL.md) | Generates practical Marathi recipes with ingredients, steps, tips, and substitutions. |
| `marathi-news-simplifier` | [`skills/marathi-news-simplifier/SKILL.md`](skills/marathi-news-simplifier/SKILL.md) | Simplifies difficult Marathi or mixed-language news into neutral, easy-to-read Marathi with summary, key points, impact, difficult terms, and a factual explanation. |

## Conventions

- Skill folders live under `skills/`
- Each skill uses a `SKILL.md` entry point
- Supporting material belongs in `references/` and `assets/`
- Skill names should match their folder name

## Notes

- The catalog above reflects the skills currently present in this repository.
- Click the linked `SKILL.md` file for each skill to read its full instructions.
